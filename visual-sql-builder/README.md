# Visual SQL Builder

A simple UI based tool to build sql queries.

---

### Requirements

Make sure you have these installed first:
* Node.js
* Python
* MySQL Engine

---

### How to Run

**Important:** You need to open **two separate terminals** for this project.

**1. Clone the Project**
```
git clone https://github.com/atharavkasture/visual_code_and_sql_editor.git
cd visual_code_and_sql_editor
```

**2. Start the Backend (Terminal 1)**
```
cd backend
npm install  # (Only need to run this the first time)
npm start
```
The server will be running on http://localhost:3002.

**3. Start the Frontend (Terminal 2)**
```
npm install  # (Only need to run this the first time)
npm run dev
```
The app will open at http://localhost:5174.

**4. Open in Browser**

Go to http://localhost:5174 to use the app.
